### Configuración del bot

Abre el [bot pvp.trade] (https://t.me/pvptrade_bot) y ejecuta /start para comenzar. ¡Luego, añádelo a un chat grupal con tus amigos para empezar!

<Frame>
  ![Set up the bot](/images/addbot.png)
</Frame>

### Depósitos

A continuación, deposita fondos en tu billetera. Puedes enviar USDC en Hyperliquid, Arbitrum o Solana. Ejecuta /start para ver las direcciones a las que puedes enviar. Debes depositar al menos 10 USDC para iniciar el proceso de transferencia.

<Frame>
  ![Deposit funds](/images/start.png)
</Frame>

### Exploración de tickers

Usa el comando /markets para ver los mercados de perps en los que puedes abrir posiciones largas o cortas.
<Frame>
  ![View Markets](/images/markets.png)
</Frame>

### Abriendo tu primera posición

Una vez que tu cuenta tenga fondos, escribe /long o /short para abrir una posición. Luego se te pedirá que elijas el ticker, el apalancamiento deseado y el margen. A continuación, una ventana emergente indicará el tamaño nocional de tu posición (Apalancamiento * Margen) y tu precio de liquidación. Selecciona confirmar para colocar la orden de mercado.

<Frame>
  ![Long HYPE](/images/long.png)
</Frame>

El chat grupal que creaste será notificado de tu operación.

### Visualización de tus posiciones

Ahora que tienes una posición abierta, puedes ver tus posiciones con el comando '/positions', ya sea en mensajes directos con el bot o públicamente en tu chat grupal.

### ¡Has terminado!

Congratulations, you've placed your first perp trade on PVP! Make sure to try out other commands like:

* `/leaderboards` — see the group leaderboard

* `/chart` — view the chart for a token

* `/stoploss` — to place a stop loss order

* `/takeprofit` — to place a take profit order

* `/track` — track a wallet's trades

* `/invite` — invite friends with your referral code

* `/points` — see how many points you have

* `/taunt` — taunt another clan with a message

And remember, don't get liquidated.
